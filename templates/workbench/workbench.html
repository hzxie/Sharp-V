    Dataset File Format

    The uploaded series-matrixing dataset file is required as a CSV formated text file that include a tab-delimited value-matrix table generated from the 'VALUE' column each sample, headed by sample. These files are suitable for loading into spreadsheet applications such as Excel.

    Metaset File Format

    The uploaded optional metadata is also required in CSV format and contains comprehensive clinical information. The same line of matrixing metadata and dataset corresponds to the same sample.
